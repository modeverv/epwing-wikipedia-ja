# CODEX_PROMPT.md

以下をCodexの最初の指示として使用できます。

---

このリポジトリでは、2026年時点の日本語Wikipediaを高機能EPWING互換辞書へ変換するv2実装を行います。

作業開始前に必ず次を順に読んでください。

1. `AGENTS.md`
2. `MEMORY.md`
3. `LOG.md`末尾
4. `CURRENT_TASK.md`
5. `TASKS.md`

最初の未完了タスクは`TASK-A001`です。最初のセッションでは`TASK-A001`だけを実装してください。ほかのタスクを先取りしないでください。

実装前に`CURRENT_TASK.md`を`TASK-A001`の内容で更新してください。

実装ルール:

- 変更は最小限にする
- テストを追加する
- Docker/hostへ不要な依存を入れない
- secretを保存・表示しない
- 完了条件を満たさない限り完了扱いにしない
- 実行したコマンドと結果を`LOG.md`へ追記する
- 長期的な設計変更を勝手に行わない
- `ARCHITECTURE.md`の境界を守る

最後に次を報告してください。

- 変更したファイル
- 実装内容
- 実行したテストと結果
- 未解決事項
- 次のTask ID
