# EDIT.md (編集・レイアウト方針)

本ドキュメントは、Wikipedia記事からEPWINGテキストへ変換する際の標準レイアウト・表記ルールを定めた仕様書です。

## 1. インフォボックス (Infobox)

Infobox は `【...】` 形式のフィールド表現で出力します。

- タイトル行: `【Infobox {title}】`（タイトルが存在する場合）
- 各フィールド: `【{項目名}|{値}】`
- 画像（EPWINGグラフィックが存在する場合）: 制御コード `\x1eG:{graphic_name}\x1f`
- 画像（フォールバック）: `【画像|{image_src}】`

例:
```text
【Infobox Album】
【作品名|日本】
【種類|スタジオ・アルバム】
【アーティスト|チューリップ】
【リリース|1975年12月1日】
```

## 2. セクション見出し (Section Headings)

セクション見出しは `■` 記号を接頭辞として出力します。

- 見出し表記: `■ {見出し名}`

例:
```text
■ 国号
```

## 3. リスト・列挙 (Lists)

箇条書きおよび順序付きリストは以下のルールでフォーマットします。

- 順序付きリスト (Ordered List): `1. {内容}`, `2. {内容}`
- 非順序付きリスト (Unordered List / サブ項目): ` ・{内容}` (インデント + 中黒)

例:
```text
1. せめて最終電車まで
 ・作詞／作曲：財津和夫　編曲：チューリップ　ボーカル：財津和夫
 ・踏切の鐘の音と電車が通り過ぎるSEから始まる楽曲である。
2. ぼくのお話
 ・作詞／作曲：財津和夫
```

## 4. 表 (Tables)

表はプレーンテキストによるグリッド（枠線 `|`）形式で描画します。

例:
```text
| 項目名 | 内容 |
| ------ | ---- |
| 東京   | 首都 |
```
